# Jester Modding

[![license](https://img.shields.io/badge/license-CC_BY--NC--ND_4.0-blue)](https://github.com/Heatblur-Simulations/jester-modding/blob/master/LICENSE)

![Jester](https://i.imgur.com/s3wAtUS.png)

Home of Jester (V2) by Heatblur Simulations.

The repository contains all existing Jester Lua logic,
allowing modders to learn how to use the Jester API and to ease **modification** of
existing behavior, as well as creation of **new functionality as mods**.

## Propose Mods

If you created a cool Jester mod or improved existing behavior and think it should make
its way into the module, just open a
[Pull Request](https://github.com/Heatblur-Simulations/jester-modding/pulls)! 🎉
